# Legacy (archived)

The original implementation: an **ASP.NET Web Forms** (.NET Framework) page where the
Framingham scoring lived entirely in client-side JavaScript (`FraminghamCalculator.js`).
It was built as an [OSCAR EMR](https://oscar-emr.com/) plugin form (note the `oscarDB`
attributes in `Site.master`).

It's kept here for reference only. The maintained app is the modern .NET + React rewrite in
the repository root — see the top-level [`README.md`](../README.md).
